# Measured-In-The-Loop Experiment Design

Design or refine an experiment plan for measured-device-aware MNSIM studies.

Use the repository context:

- measured data comes from `test_data/`
- preset extraction is already supported by project scripts
- formal-space and guidance-space workflows already exist

Tasks:

- identify the exact experiment objective
- propose the smallest useful run plan first
- specify seeds, budget, network, config source, and accuracy settings when relevant
- separate pilot runs from paper-scale runs
- suggest artifact locations consistent with the repo structure

Output format:

- objective
- required inputs
- run plan
- success criteria
- risks or confounders
- next-step recommendation
