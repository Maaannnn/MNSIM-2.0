# Research Prompt Library

This prompt library adapts the `decision-grade-memory` workflow idea to `MNSIM-2.0`.

Use these prompts as starting templates for Codex sessions. They are meant to reduce setup time, improve continuity, and keep experiment, analysis, and writing work clearly separated.

## Prompt groups

- `setup/`: initialize a focused Codex session for this repository
- `literature/`: perform broad literature search, connect experiments to paper positioning, and strengthen related work
- `experiment_design/`: design or refine experiment matrices and search studies
- `analysis/`: summarize outputs, compare runs, and extract paper-ready findings
- `writing/`: draft or revise sections without overstating claims

## Recommended order

1. start with `setup/project_session_bootstrap.md`
2. use a literature prompt early if the task involves positioning, baselines, or related work
3. use an analysis prompt after artifacts exist
4. use a writing prompt when results are ready to be turned into prose
