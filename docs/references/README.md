# 参考文献与外部资料

这个目录存放外部参考材料，包括调研报告、开题文档和官方手册。这些文件不是实验执行或论文草稿，而是研究背景的参考资料。

| 文件 | 类型 | 说明 |
|------|------|------|
| `MNSIM_manual.pdf` | 官方手册 | MNSIM-2.0 官方使用手册（IEEE TCAD 2023 配套） |
| `【深度调研】RRAM CIM非理想性对PPA设计的影响及优化策略.pdf` | 调研报告 | RRAM CIM 非理想性综合调研，其 markdown 摘要见 `docs/simulator/aime_mnsim.md` |

| `开题报告v3-11.14.docx` | 开题报告 | 硕士论文开题报告（v3） |
| `深度调研：器件非理想特性对RRAM CIM PPA最优设计的影响与仿真框架选型分析.docx` | 调研报告 | 器件非理想特性与框架选型分析（早期版本，已被后续文档supersede） |
