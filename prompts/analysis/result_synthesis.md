# Result Synthesis Prompt

Summarize one or more completed experiment runs from this repository.

Requirements:

- name the exact run directories used
- separate observed results from interpretation
- call out missing seeds, partial coverage, or missing accuracy evaluation
- highlight only findings supported by artifacts
- mention what further run would most reduce uncertainty

Desired output:

- short execution summary
- key findings
- limitations
- paper-ready takeaway sentences
- follow-up experiment recommendation
